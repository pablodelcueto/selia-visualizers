# ProgramResources

WebGL Program resources object. Objects of this type contain all the webGL references required in the program, including textures, buffers and locations.

Properties:
Name Type Attributes Description
program WebGLProgram

The webGL program.
buffers Object.<string, WebGLBuffer>

An object containing all buffers required in the program.
locations Object.<string, (number|WebGLUniformLocation)>

An object containing all the locations required in the program.
textures Object.<string, WebGLTexture> <optional>

An object containing all the textures required in the program.
